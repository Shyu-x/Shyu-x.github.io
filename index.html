03. 个人项目

FEATURED PROJECT

core-sh: A POSIX-compliant Shell

一次对操作系统核心的深度探索。`core-sh` 不仅仅是命令的执行者，更是赋予命令行生命、揭示用户空间与内核交互奥秘的实践。它解构了指令与系统调用之间的复杂舞蹈，是对进程、文件与信号机制的精细解剖。

  • 进程控制
  • I/O 重定向
  • 管道
  • 信号处理
  • 后台任务
  • 命令补全
/bin/core-sh

核心架构

命令解析器 (Parser)

将用户输入的字符串分解为结构化的命令、参数和重定向操作。

执行引擎 (Executor)

通过 `fork` 和 `exec` 系列函数创建子进程，并精确管理其生命周期。

作业控制 (Job Control)

实现前后台进程组的管理，允许任务的暂停、继续和终止。

信号处理器 (Signal Handler)

优雅地捕获并响应如 `SIGINT`, `SIGTSTP` 等异步事件，保证Shell的健壮性。

FEATURED PROJECT

nexus-net: Event-Driven Network Library

一次对高性能网络编程范式的极致追求。`nexus-net` 是一个基于 `Proactor` 模式的事件驱动网络库，是异步I/O能力的证明。它利用 Linux `epoll`，将网络事件的检测、分发和处理流程化，旨在提供一个低延迟、高吞吐、可伸缩的服务器端编程框架。

  • Proactor 模式
  • Epoll
  • 非阻塞 I/O
  • 线程池
  • 零拷贝
  • 定时器

核心组件 & 性能亮点

事件循环 (Event Loop)

库的核心脉搏，通过 `epoll_wait` 高效监听并分发 I/O 事件。

多核友好线程池

将计算密集型业务逻辑与 I/O 线程分离，充分利用多核 CPU 资源。

高效缓冲区 (Buffer)

设计的可自动扩容的缓冲区，有效减少内存拷贝和系统调用次数。

毫秒级定时器

基于时间轮或最小堆实现，用于管理大量连接的超时事件。
